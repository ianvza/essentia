# Essentia
A modern, responsive web application dedicated to educating users on the safe, holistic, and evidence-backed uses of essential oils for daily healthcare and wellness.

Welcome to **EssenceHealth**, a comprehensive, user-friendly web platform designed to bridge the gap between traditional wellness and modern digital accessibility. This website serves as a guide for individuals looking to integrate essential oils safely into their daily healthcare and wellness routines.

## ✨ Features

* **Oil Directory:** An interactive library detailing various essential oils (e.g., Lavender, Peppermint, Tea Tree), their extraction methods, primary benefits, and aromatic profiles.
* **Symptom Checker / Guide:** A search tool where users can find recommended oils based on common wellness goals (e.g., stress relief, sleep quality, muscle recovery, focus).
* **Safety & Dilution Calculator:** A built-in calculator or guide ensuring users understand safe dilution ratios with carrier oils for topical application.
* **DIY Recipes:** A curated collection of safe, easy-to-follow recipes for diffusers, roll-ons, and natural skincare.
* **Responsive Design:** Fully optimized for smooth navigation across desktops, tablets, and smartphones.
